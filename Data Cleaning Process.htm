Data Cleaning Process

    The data cleaning process was done on Power BI power query, The dataset was first pulled into the power query using Get data on the Power BI home tab, and the 12 months of data merged. The following cleaning process was then carried out.

  • Created a new custom column 'ride_length' by subtracting the 'started_at' from the 'ended_at'
  • Inserted a 'date' column
  • Filtered out the empty cell in 'end_lng'
  • Created a 'date' column using the 'started_at' column as sample
  • Removed 'Source.Name', 'start_station_name', 'start_station_id', 'end_station_name', 'end_station_id', and 'day_of_week' columns

    • The dataset was then loaded into the Power BI environment.
